# Analisis Data Bisnis (Latihan)

Project ini merupakan latihan analisis data menggunakan Python (Pandas) dan SQL.

## Tujuan Analisis
Menganalisis data penjualan untuk mendapatkan insight bisnis seperti:

- Total penjualan
- Produk paling laris
- Distribusi kategori produk
- Pola transaksi pelanggan

## Dataset
Dataset yang digunakan:

- data-latihan-kotor.csv → data mentah
- data-latihan-bersih → data setelah proses cleaning

## Proses Analisis

Tahapan analisis yang dilakukan:

1. Data Cleaning
2. Data Transformation
3. Exploratory Data Analysis (EDA)
4. Insight Bisnis

## Tools yang Digunakan

- Python
- Pandas
- Jupyter Notebook
- SQL

## File dalam Repository

| File | Deskripsi |
|-----|------|
| 01_latihan-analisis.ipynb | Notebook analisis data |
| data-latihan-kotor.csv | Dataset mentah |
| data-latihan-bersih | Dataset setelah cleaning |
| latihan.sql | Query SQL untuk analisis |

## Insight yang Didapat

Contoh insight:

- Produk kategori A memiliki penjualan tertinggi
- Sebagian besar transaksi berasal dari kategori B
